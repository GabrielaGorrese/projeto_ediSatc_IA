## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema baseado em **Inteligência Artificial** capaz de **gerar resumos, análises e categorizações automáticas de editais públicos**, com foco em chamadas de órgãos como **FAPESC** e **FINEP**.

A solução tem como objetivo auxiliar instituições de ciência e tecnologia, como o **Centro Tecnológico SATC**, a otimizar o processo de análise de editais, reduzindo esforço manual e minimizando erros.
